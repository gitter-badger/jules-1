jules
=====

jules is a statistical computing library that aims to provide R-like experience in modern C++.

Dependencies
------------

 - C++14 compiler
 - [debug_assert](https://github.com/foonathan/debug_assert)
 - [range-v3](https://github.com/ericniebler/range-v3)
